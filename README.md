# Imaging_fault_system_using_2D_electrical_resistivity_tomography_ERT_Thesis
The thesis presents an integrated investigation of the Swabian Fault System in the Ammer Valley (southwestern Germany) using Electrical Resistivity Tomography (ERT), with emphasis on data processing, inversion, geostatistical regularization, synthetic modelling, and Smoothness-Constrained Geometry-Preserving Inversion (SCGPI).

This thesis is supported by the following repositories, which contain the raw data, processing workflows, inversion scripts, and modelling codes used throughout the study:

1_ERT_Raw_Data
Raw ERT datasets acquired at the Wendelsheim and Pfäffingen field sites.

2_ERT_Data_Processing
Python scripts for raw data visualization, pseudosection generation, filtering workflows, L-curve-like analysis, and filtered pseudosection plotting.

3_ERT_Inversion
pyGIMLi inversion workflows, including classical smoothness-constrained inversion, geostatistical regularization, cross-array forward modelling, and starting model implementation.

4_ERT_Modeling
Synthetic modeling and Smoothness-Constrained Geometry-Preserving Inversion (SCGPI) scripts used to investigate different fault representations and evaluate inversion performance.

Together, these repositories provide the complete workflow of the study, from field data acquisition and preprocessing to inversion, modeling, interpretation, and the final thesis.
